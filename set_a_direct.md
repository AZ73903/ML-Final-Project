# Question Set A — Structured Preference (Waive Builder)

**Philosophy:** All multiple-choice or recognition-scale questions. Workers select from options — no free text. Fast to answer, but prone to social desirability bias and lazy/dishonest responses. Sourced from the Waive Builder intake form (project-vigil).

**Format:** Every question is structured. Responses are one of the provided options (or Yes/Somewhat/Not really for Section 6).
